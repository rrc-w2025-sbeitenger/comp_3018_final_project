Admin

Set Firebase custom claims for a user

Authorizations:
bearerAuth
Request Body schema: application/json
required
uid
required
string

The Firebase user ID of the user receiving the custom claims.

required
object (AuthorizationOptions)

Responses

Local server

http://localhost:3000/api/v1/admin/setClaims

Request samples

  • Payload
Content type
application/json
{
  • "uid": "abc123FirebaseUid",
  • "claims": {
    • "hasRole": [
      • "admin"
      ],
    • "allowSameUser": true
    }
}

Response samples

  • 200
  • 401
  • 403
Content type
application/json
"Custom claims set for user: abc123FirebaseUid. User must obtain a new token for changes to take effect."
